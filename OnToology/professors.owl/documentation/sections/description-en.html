The classes and properties defined by this module are represented in the following UML: ![Module UML](resources/image1.png) As it can be seen in the diagram, a relation between the custom classes and properties and some classes and properties defined by [FOAF](http://xmlns.com/foaf/spec/) is established. By using this schema, and in combination by the courses defined in [OntoCIS Courses](../../courses.owl/documentation/index-en.html), we can describe all the information for each professor.
