Introducción volver a índice

Este módulo incluye aquellas relaciones y clases necesarias para modelar las competencias y objetivos de aprendizaje. * **Competencia:** una frase que describe de manera general el conocimiento y capacidades que debe adquirir un estudiante. * **Objetivo de aprendizaje:** una frase que describe exactamente lo que un estudiante podrá hacer de manera mensurable/evaluable. La diferencia clave entre ellos, es que un objetivo debe ser presentado de forma tal de que pueda ser medido o evaluado. Normalmente, un objetivo de aprendizaje será una versión más específica de una competencia. Por ejemplo, una competencia puede ser “la capacidad para implementar algoritmos y estructuras de datos en el software”, mientras que, un objetivo de aprendizaje puede ser “implementar algoritmos numéricos básicos”.
